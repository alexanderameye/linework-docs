    Troubleshooting and Known Limitations

    Troubleshooting and Known Limitations

    Aug 07, 20242 min read

    Some frequently encountered issues and how to resolve them. Can’t find a solution to your issue here? Contact me.

    Known Limitations (fixes in progress)

    Soft/Wide Outline + MSAA

    Having Anti Aliasing (MSAA) enabled in the Universal Render Pipeline Asset will result in an error and the outlines will not render.

    This will be fixed in a future update.

    Alpha cutout

    Alpha cutout is not currently supported but support will be added soon. See the Roadmap.

    VR support

    VR is not currently supported but support will be added soon. See the Roadmap.

    Sprite support

    Sprites are not currently supported but support will be added soon. See the Roadmap.

    Known Limitations (by design)

    Temporal Anti-aliasing (TAA)

    If the outline is set to render After Post Processing, the outline will jitter more than the rest of the scene.

    Volume System

    This asset has not been designed to work together with the URP Volume System.

    Deferred Rendering

    Linework is not known to be compatible with the deferred rendering mode.

    Particles and transparent objects

    Adding outlines to particles or other transparent objects is not supported.

    Compatibility

    See Compatibility for more information.

    General

    Outline/fill does not behave as expected

    When combining multiple types of outline effects or fills, it could sometimes be that their stencil masks interact with each other. Enable the Force Clear Stencil toggle in the settings to see if this resolves the issue. If you encounter an issue like this, please Contact me since I treat it as a bug.

    Outline/fill has influence on other rendering effects

    Please Contact me.
